| Agent                         | Primary Function                                                                                           | Inputs                                                   | Outputs                                                                                           |
|--------------------------------|-----------------------------------------------------------------------------------------------------------|----------------------------------------------------------|---------------------------------------------------------------------------------------------------|
| **Orchestration Agent (OA)**  | Coordinates the workflow across all modular agents, ensures tasks are executed in sequence and integrated. | Educator’s input (syllabus, free-form text, or voice)    | Triggered tasks sent to appropriate agents; combined final outputs.                              |
| **Curriculum Validator Agent (CV)** | Analyzes educator’s curriculum to identify gaps, overlaps, and missing AI concepts.                       | Curriculum documents, syllabus, or uploaded text         | Curriculum gap matrix (Excel/CSV), highlighting covered vs. missing content.                     |
| **Quiz Creator Agent (QC)**   | Designs quizzes aligned with curriculum and learning objectives.                                          | Validated curriculum, educator preferences               | Quizzes formatted as slide blocks (PowerPoint), or embedded in interactive modules.              |
| **Content Enricher Agent (CE)** | Suggests additional reading material, case studies, and enrichment resources.                             | Curriculum content, reference databases                  | “Click to Expand” modules with external resources, suggested readings, or interactive extensions.|
| **Feedback Evaluator Agent (FA)** | Assesses the quality of generated quizzes/content and refines based on educator feedback.                  | Educator review, learner engagement signals              | Iterative improvements to quizzes, content, and enrichment material.                             |
| **Memory Agent (MA)**         | Stores past educator inputs, preferred quiz styles, and content usage for reuse and personalization.      | Historical interactions, prior curriculum reviews        | Personalized suggestions, continuity across multiple curriculum reviews.                         |
